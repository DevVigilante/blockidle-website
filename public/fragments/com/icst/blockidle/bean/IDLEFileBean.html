IDLEFileBean

Represents a base file in the Block IDLE file system. This class serves as a base model for all file-like structures in the system and provides basic metadata such as the file name.

This class implements Serializable to allow instances to be serialized and passed between components or stored persistently.

Methods

getFileType

Returns the file type associated with this file.

This method can be overridden by subclasses to provide specific file types.

  • return: the file type as a string, or an empty string if undefined
