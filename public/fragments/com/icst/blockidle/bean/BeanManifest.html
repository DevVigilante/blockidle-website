BeanManifest

Represents a container for a list of BeanMetadata instances.

This class supports retrieving all metadata entries or filtering them by a specific subtype. Implements Serializable for object serialization support.

Methods

getMetadata

Returns the list of all metadata entries stored in this manifest.

  • return: a list of BeanMetadata instances, or null if not initialized

setMetadata

Sets the list of metadata entries for this manifest.

  • metadata: BeanMetadata> — the list of BeanMetadata to assign

get

Returns a list of metadata elements that are instances of the specified subclass type.

  • T: Unknown — the type of metadata to filter for

  • classType: Class<T> — the Class object representing the desired type

  • return: a list containing only the metadata objects of the specified type
