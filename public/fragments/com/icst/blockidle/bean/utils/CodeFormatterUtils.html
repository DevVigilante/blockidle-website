CodeFormatterUtils

Utility class providing methods for formatting and analyzing code strings, such as generating syntax keys, applying indentation, and calculating indentation levels.

This class is not intended to be instantiated or extended.

Methods

getKeySyntaxString

Formats the given key as a custom syntax string in the format <CodeKey : key> which is used by Block IDLE application.

  • key: String — the key to be wrapped in the syntax format

  • return: a formatted string containing the given key

addIntendation

Adds tab-based indentation to each line of the given code string.

The first line is kept as-is, and all subsequent lines are prefixed with the specified number of tab characters. Read getIntendation to know why intendation is nog applied to first line.

  • code: String — the multi-line code string to format

  • intendation: int — the number of tab characters to prepend to each line (except the first)

  • return: the indented code string

getIntendation

Calculates the indentation (leading space count) of the first line in the given code that contains the specified item.

  • codeSyntax: String — the full code block to search through

  • item: String — the string to locate in the code

  • return: the number of leading spaces (indentation level) before the item, or 0 if the item is not found
