BeanArrayCloneUtils

A utility class providing the deep clone functionality for followings:

  • ArrayList of bean that implements CloneableBean
  • Array of DatatypeBean.

Methods

clone

Creates a deep copy of the given ArrayList containing CloneableBean instances.

This method is designed to safely handle corner cases, including null input. If the input list is null, the method will return null instead of throwing an exception.

  • T: Unknown — the type of elements in the ArrayList that implements CloneableBean

  • beans: ArrayList<T> — the ArrayList of beans to clone; may be null

  • return: a new ArrayList containing cloned instances of the input beans, or null if the input list is null

cloneDatatypeBeanArray

Creates a deep copy of the given array of DatatypeBean instances.

This method is designed to handle corner cases gracefully, including null input. If the input array is null, the method returns null instead of throwing an exception.

  • datatypeBean: DatatypeBean[] — the array of DatatypeBean instances to clone; may be null

  • return: a new array containing cloned instances of the input beans, or null if the input array is null
