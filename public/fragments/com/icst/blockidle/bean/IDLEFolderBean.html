IDLEFolderBean

Represents a folder in the Block IDLE file system. This class extends IDLEFileBean and provides a specific implementation for folder-type files.

This class implements Serializable to allow instances to be serialized and passed between components or stored persistently.

Methods

getFileType

Returns the file type of this object, which is always "folder".

  • return: a string representing the file type, i.e., "folder"
