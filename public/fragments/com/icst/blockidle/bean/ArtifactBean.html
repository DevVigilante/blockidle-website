ArtifactBean

Represents a Maven-style artifact with metadata such as group ID, artifact ID, version, repository location, and file extension.

This bean is serializable and extends BeanMetadata.

Methods

getGroupId

Returns the group ID of the artifact (e.g., "com.example").

  • return: the group ID

setGroupId

Sets the group ID of the artifact.

  • groupId: String — the group ID to set

getArtifactId

Returns the artifact ID of the artifact (e.g., "my-library").

  • return: the artifact ID

setArtifactId

Sets the artifact ID of the artifact.

  • artifactId: String — the artifact ID to set

getVersion

Returns the version of the artifact (e.g., "1.0.0").

  • return: the version string

setVersion

Sets the version of the artifact.

  • version: String — the version to set

getRepositoryName

Returns the name of the repository hosting the artifact.

  • return: the repository name

setRepositoryName

Sets the name of the repository hosting the artifact.

  • repositoryName: String — the repository name to set

getRepositoryUrl

Returns the URL of the repository where the artifact is located.

  • return: the repository URL

setRepositoryUrl

Sets the URL of the repository where the artifact is located.

  • repositoryUrl: String — the repository URL to set

getExtension

Returns the file extension of the artifact (e.g., "jar", "aar").

  • return: the artifact's file extension

setExtension

Sets the file extension of the artifact.

  • extension: String — the file extension to set
